This library is a small set of algorithms and data processing
utilities for [[https://en.wikipedia.org/wiki/Natural_language][Natural Languages]].

- Features
  - simple algorithms for:
    - tokenization
      - token storage (rocksdb)
    - stemming (porter)
    - TODO lemmatization
    - data clustering (dbscan)
